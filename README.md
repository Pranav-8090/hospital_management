# 🏥 Hospital Management System

## 📌 Project Description
This is a Java-based Hospital Management System that helps manage doctors, patients, and appointments efficiently using Object-Oriented Programming and Collection Framework.

---

## 🚀 Features
- Add Doctor
- View Doctor Details
- Add Patient
- View Patient Details
- Book Appointment
- Manage Hospital Records

---

## 🛠️ Technologies Used
- Java (Core Java)
- OOP Concepts (Encapsulation, Inheritance, Polymorphism)
- Collection Framework (ArrayList, etc.)

---

## 📂 Project Structure
- `Doctor.java` → Doctor entity class  
- `Patient.java` → Patient entity class  
- `Appointment.java` → Appointment details  
- `Person.java` → Base class  
- `DoctorService.java` → Doctor operations  
- `PatientService.java` → Patient operations  
- `HospitalService.java` → Service interface  
- `HospitalApp.java` → Main class  

---

## ⚙️ How to Run
1. Clone the repository  
2. Open in Eclipse / IntelliJ  
3. Run `HospitalApp.java`
